# XP — Dashboard

Base: `https://api.ethos.network/api/v2/xp`

All dashboard endpoints use `profileId` (integer).

## GET `/xp/dashboard/yield/{profileId}`

Yield data for a profile's XP dashboard.

## GET `/xp/dashboard/periodic-rewards/{profileId}`

Periodic rewards for a profile.

## GET `/xp/dashboard/multipliers/{profileId}`

XP multipliers for a profile.

## GET `/xp/dashboard/daily-earnings/{profileId}/{date}`

Daily earnings for a profile on a specific date.

## GET `/xp/dashboard/daily-awards/{date}`

Daily awards for a specific date (global).

## GET `/xp/dashboard/weekly-delegator-award/{profileId}`

Weekly delegator award for a profile.
