# Chains — Manage

Base: `https://api.ethos.network/api/v2`

## GET `/chains`

List all available chains.

## POST `/chains` 🔒 (admin)

Add a new chain.

## PUT `/chains/{id}` 🔒 (admin)

Edit an existing chain.

## DELETE `/chains/{id}` 🔒 (admin)

Delete a chain.

## GET `/chains/gas-price` (beta)

> Beta — may change without notice.

Get current Base gas price.
